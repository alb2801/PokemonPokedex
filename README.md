# Pokédex & Guess the Pokémon

Aplicación web desarrollada con Angular que consume la PokéAPI para visualizar información de Pokémon y jugar a adivinarlos.

## Descripción

Este proyecto es una Pokédex interactiva que permite:

- Buscar Pokémon por nombre  
- Ver información detallada (stats, habilidades, tipos)  
- Jugar un mini juego de adivinanza  

Incluye un modo **"Guess the Pokémon"**, donde el usuario debe identificar un Pokémon a partir de su silueta.

---

## Guess the Pokémon

- Se muestra un Pokémon en forma de sombra  
- El sprite utilizado es **shiny**  
- Puede ser de **cualquier generación**  
- El usuario ingresa el nombre para adivinar  
- Sistema de puntaje básico  

---

## Tecnologías
